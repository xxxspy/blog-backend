根据一列名称来创建一对工作表，工作表的名称就是这一列数据中的数据，这该怎么办？这里分享一个快速创建的方法，不使用任何vba代码，手工操作就可以完成。

- 打开Excel，选中我们工作表名称这一列数据

{% asset_img bd70426034a85edf6e2c502f48540923dc5475df.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

- 在插入选项下，找到数据透视表，我们点击它

{% asset_img b9d8b7014c086e067877914703087bf40bd1cba4.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

- 打开了一个对话框，我们看到选中的区域已经添加到了对话框中，直接点击确定按钮

{% asset_img a84052086e061d956c9999347af40ad163d9caa4.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

- 在工作表的右侧出现了一个对话框，如图所示，首选勾选报表名称，这是我们选择的数据的第一个单元格的值

{% asset_img 1cd4147b02087bf46333f9e5f3d3572c10dfcfa4.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

- 在分析选项下找到【选项】菜单，如图所示，在选项菜单中，点击【显示报表筛选页】

{% asset_img 109eb7ec8a13632713326c3f908fa0ec09fac7a4.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

{% asset_img bd7faf3533fa828b508333effc1f4134960a5adf.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

{% asset_img a1ad16fa513d2697210ca65154fbb2fb4216d8a4.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

- 出现了一个对话框，直接点击确定按钮

{% asset_img 58af236d55fbb2fb3ff4f61e4e4a20a44723dca4.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

- 我们看到，工作表已经创建好了，接着设置工作表的格式，如图所示

{% asset_img 6c63514a20a4462388150b6e9922720e0df3d7a4.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

- 将列标题复制下载

{% asset_img 476217f790529822f6ec97cfd6ca7bcb0b46d4a4.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

- 选中所有的单元格，方法是使用shift键，点击【北京】和【sheets5】，这样之间的工作表就选中了

{% asset_img c87c6ecf3bc79f3dff424a3cbba1cd11738b29df.jpg Excel2013教程-快速创建大量指定名称的工作表 %}

- 将复制好的内容粘贴到【北京】工作表，这样所有的被选中的工作表中都粘贴了这个列标题，这样具有相同格式的大量工作表就创建完成了

{% asset_img f99dcf00baa1cd117915dc2cb812c8fcc2ce2ddf.jpg Excel2013教程-快速创建大量指定名称的工作表 %}
