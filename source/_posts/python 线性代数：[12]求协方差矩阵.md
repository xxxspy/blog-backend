- 这是今天用到的所有代码：

- >>> import numpy

- >>> s=[100,120,140]

- >>> t=[50,60,70]

- >>> y=s+t

- >>> y

- [100, 120, 140, 50, 60, 70]

- >>> y=[s,t]

- >>> y

- [[100, 120, 140], [50, 60, 70]]

- >>> a=numpy.cov(y)

- >>> a

- array([[ 400.,  200.],

-        [ 200.,  100.]])

- >>> 

- >>> x=[1,0,1]

- >>> y=[s,t,x]

- >>> 

- >>> a=numpy.cov(y)

- >>> a

- array([[  4.00000000e+02,   2.00000000e+02,   0.00000000e+00],

-        [  2.00000000e+02,   1.00000000e+02,   0.00000000e+00],
